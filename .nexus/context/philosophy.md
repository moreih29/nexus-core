# nexus-core 철학

## 1. 정체성

nexus-core는 3 하네스(Claude Code, OpenCode, Codex)의 Nexus 플러그인을 위한 공통 라이브러리다.

## 2. 두 공통화 축

### 2-1. 에이전트 조율 계층

에이전트를 HOW / DO / CHECK 3종으로 분류하고, Lead가 이들을 주재한다.

### 2-2. 공통 빌드 도구

각 하네스의 스킬/툴/훅 구조 중 공통화 가능한 부분을 단일 기준 변환으로 제공한다. 이를 통해 3 하네스 간 구현 편차를 방지하고 동일 철학이 일관되게 전파되도록 한다.

## 3. 경계

실제 플러그인 빌드 및 배포는 개별 하네스 책임. nexus-core는 공통 자산 + 빌드 유틸리티만 제공한다.
