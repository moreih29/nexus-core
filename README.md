# nexus-core

3 하네스(Claude Code, OpenCode, Codex)의 Nexus 플러그인을 위한 공통 라이브러리.

## 제공하는 것

1. 에이전트 조율 계층 — HOW / DO / CHECK 분류 + Lead 주재
2. 공통 빌드 도구 — 하네스의 스킬/툴/훅 구조 중 공통화 가능한 부분을 단일 기준 변환으로 제공

## 제공하지 않는 것

실제 플러그인 빌드 및 배포는 개별 하네스 책임.

## Consumer 하네스

- claude-nexus (Claude Code)
- opencode-nexus (OpenCode)
- codex-nexus (Codex)
